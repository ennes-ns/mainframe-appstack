# Nextcloud Stack Mandates

This directory contains the Nextcloud service stack, including MariaDB and Redis.

## Service Details
- **Nextcloud:** lscr.io/linuxserver/nextcloud
- **Database:** lscr.io/linuxserver/mariadb
- **Cache:** redis:alpine
- **Public URL:** https://cloud.nschouteren.nl
- **Private URL:** http://100.95.98.74:12345 (Tailscale only)

## Secret Management
- **Location:** `./secrets/`
- **Secrets Protocol:** NEVER read the content of files in the secrets directory.
- **Initialization Issue:** Automated installation via `occ` failed due to database connection issues during initialization. Manual setup via the web interface is required for the first run.

## Setup Instructions
1. **Stop Traefik:** `docker stop traefik` (to prevent public access during setup).
2. **Access Setup:** Navigate to `http://100.95.98.74:12345` via Tailscale.
3. **Database Configuration:**
   - **User:** `nextcloud`
   - **Password:** (From `./secrets/db_password.txt`)
   - **Database Name:** `nextcloud`
   - **Host:** `nextcloud-db`
4. **Finalize:** Complete the admin account creation.
5. **Restart Traefik:** `docker start traefik`.

## Troubleshooting
- **Internal Server Error (500):** Usually caused by a failed database handshake. Ensure the `nextcloud-db` container is healthy and the credentials match the secret files.
- **Permissions:** Files in `data/` and `db/` are owned by `PUID 1000 / PGID 1000` (user `ennes`).
